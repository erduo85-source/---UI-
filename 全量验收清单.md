# 国内游卡账号中心全量验收清单

验收日期：2026-05-25

验收依据：

- 测试用例：`测试用例.md`
- 需求文档：`国内账号中心V1.5.0-需求文档.md`、`国内账号中心V1.5.1-需求文档.md`
- 设计稿：[`国内 游卡账号中心`](https://www.figma.com/design/zdbqXSiRNRL373XDbQSnht/%E5%9B%BD%E5%86%85-%E6%B8%B8%E5%8D%A1%E8%B4%A6%E5%8F%B7%E4%B8%AD%E5%BF%83?node-id=42603-155)

验收口径：

- 状态仅使用：`通过`、`不通过`、`部分通过`、`未验证`。
- UI 验收采用桌面单视口分级判定，检查布局、间距、字号、颜色、圆角、阴影、按钮状态、弹窗状态等；不使用机器逐像素 diff 作为唯一结论。
- 当前账号在本轮复测时为：已登录、已实名、已绑定登录手机、已绑定安全手机、未绑定安全邮箱、已设置登录密码。
- 历史截图证据来自同一验收会话中已保存的页面状态；当前账号状态已变化的用例会在备注中说明。
- 允许点击发送验证码，但不提交绑定、换绑、修改密码、重置密码等会改变账号状态的最终确认。
- 三期能力不纳入本轮验收，包括登录保护、登录设备、登录锁定、安全防护等。
